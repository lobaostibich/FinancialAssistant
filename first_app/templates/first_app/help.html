Para tornar mais fácil para você inserir dados na plataforma, vamos iniciar cadastrando algumas informações, como mostra o exemplo abaixo.

Nome Categoria Fixo ou variável? Valor Deletar Linha


Dicas:
Deletar Linha
Para deletar algum dado, basta marcar a caixinha na linha do dado que se deseja excluir, na coluna "Deletar Linha", clicar no botão "Salvar Registros" e depois clicar em "Concluir Inserção"
Fixo ou variável?
Colocando um registro como fixo, ele aparecerá pronto para você na próxima vez que for inserir dados novos
Salvar Registros
Ao clicar em salvar registros, os registros são processados para que você possa visualizar os dados no dashboard e uma nova linha é criada para que caso você queira incluir um novo dado, basta preencher essa linha, se não quiser preencher mais nenhuma informação basta clicar no botão "Concluir Inserção".
Concluir Inserção
Ao clicar no botão "Concluir Inserção" você já pode ir para a visualização de dados, onde poderá controlar com detalhes suas finanças pessoais
